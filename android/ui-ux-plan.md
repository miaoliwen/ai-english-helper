# AI英语解题助手 — UI/UX 优化计划

## 来自 UI UX Pro Max 的设计智能

| 维度 | 推荐 | 
|------|------|
| **产品匹配** | Education + AI/Chatbot Platform 混合 |
| **推荐风格** | Claymorphism + AI-Native UI |
| **颜色** | Indigo #4F46E5 + AI Purple #7C3AED + Energy Orange #F97316 |
| **版式** | 友好、清晰层次、大号正文 (≥16sp) |
| **UX 重点** | 流式输出动画、加载骨架屏、错误反馈、空状态 |
| **反模式** | 暗色模式缺省、复杂术语、无反馈空白加载 |

## 实施清单

1. 更新全局色彩令牌 (colors.xml)
2. 创建流式光标 Drawable 动画 (blinking cursor)
3. 创建 shimmer 骨架屏效果
4. 优化结果页: 流式状态栏 + 光标 + 完成态渐入
5. 优化设置页: 分组卡片 + 对比度 + 图标
6. 优化主页: 功能卡片悬浮反馈
7. 优化历史页: 空状态插图
